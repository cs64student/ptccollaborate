HTML

HTML is a markup language that serves as the backbone of all websites. A simple, basic website can be built with only HTML. As HTML serves mainly to describe and "mark up" the desired content in a semantic (meaningful) way to the intended user's browser so that the content can be displayed accordingly. CSS

CSS is a next logical "layer" to apply to your HTML website. CSS, like HTML is not technically a programming language but it follows it's own syntax and allows developers to control the presentation and style of their HTML content and elements. CSS allows web developers to not only style web-pages to look great and present information in a user-friendly way but also to add some really cool effects and animations.
